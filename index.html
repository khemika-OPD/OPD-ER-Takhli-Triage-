🚨 ระดับ 1: ภาวะคุกคามชีวิต?

มีอาการดังต่อไปนี้หรือไม่:

  • CPR / ET Tube / ICD insertion
  • GCS < 8 หรือ O2 sat < 90%
  • Shock (SBP < 90 / MAP < 65)
  • ชักเกร็ง / BP >= 220/130
  • NEW Score >= 5 (หรือข้อเดียว >= 3)

💓 ระดับ 2: ภาวะเสี่ยงสูง?

  • Fast track Stroke/STEMI
  • เจ็บหน้าอก / แขนขาอ่อนแรง (ใน 24 ชม.)
  • จิตเวชคลุ้มคลั่ง/เสี่ยงฆ่าตัวตาย
  • สัตว์พิษกัด / ได้รับสารพิษ
  • Pain Score >= 7 + V/S ผิดปกติ

🟡 ระดับ 3: ผล Lab ผิดปกติ?

พบค่าใดค่าหนึ่งดังนี้หรือไม่:

FBS ≤ 55 (non-DM)
FBS ≥ 400
Na ≤ 120 / ≥ 160
K ≤ 2.5 / ≥ 6
Hct ≤ 20%
Platelet วิกฤต
INR ≥ 5 / VCT ≥ 20
WBC ≤ 2k / ≥ 30k

📦 ระดับ 4-5: กิจกรรม/ทรัพยากร

ต้องใช้ Resource หรือไม่?

Resource: Lab, X-ray, ฉีดยา, ทำแผลซับซ้อน, สวนปัสสาวะ
